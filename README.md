# Password_Generator
It is a random password generating site that can provide you with a completely random and secure password which you can copy on click with a bit of generation options to choose from.

# Theme
It belongs to web playable endless theme because you can generate almost infinite number of passwords(3.004 x 10^39 to be precise) endlessly. You can copy whichever one you like with so many options to choose from.

# Features
- Two random password generated at once
- Copy on click
- Ability to set password length(4-20)
- Abilty to include/exclude uppercase,symbols and numbers
- Clean interface with instant generation

# Technologies used
- HTML5
- CSS3
- Vanilla JS

# Limitations
- Unable to store passwords
- Less features compared to other refined sites
